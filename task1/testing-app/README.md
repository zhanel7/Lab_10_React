В данном проекте реализовано React-приложение "Список задач (Todo List)" с использованием TypeScript.

Также добавлено модульное тестирование с помощью:
- Jest
- React Testing Library

---


Приложение поддерживает:

- Добавление задачи
- Добавление задачи по нажатию Enter
- Удаление задачи
- Переключение статуса (выполнено / не выполнено)
- Счётчик задач (всего и выполненных)
